Лабораторная работа №6. Игра «Виселица»
Как запустить программу
Выберите подходящий способ в зависимости от того, установлен ли у вас Git:
Способ 1 (если на компьютере нет Git):
Скачайте ZIP-архив из этого репозитория.
Распакуйте его в любую удобную папку.
Откройте папку в удобной для вас среде разработки (IDE).
Запустите файл Game_Hangman_Group_IT-8.py.
Наслаждайтесь игрой!
Способ 2 (если на компьютере есть Git):
Введите в Git Bash команду:
git clone https://github.com/EmptyQwazar/Game-Hangman-Group-IT-8-
Перейдите в папку репозитория:
cd Game-Hangman-Group-IT-8-
Запустите игру «Виселица»:
python Game_Hangman_Group_IT-8.py
Наслаждайтесь крутой игрой!
