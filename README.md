Лабораторная работа №6. Игра "Виселица". Работал над репозиторием и его содержимым Антипин Дмитрий Константинович, 1 курс, ИКНТ, ИТ-8 
Чтобы запустить программу можно выбрать один из пунктов запуска программы:
Пункт 1.(Если на компьютере нету Git): 
      1) Скачать ZIP файл из данного репозитория
      2) Рапсаковать в любую удобную папку
      3) Открыть папку в удобном для вас приложении для кодинга программ
      4) Запустить файл Game_Hanmgan_Group_IT-8.py
      5) Радоваться жизни
Пункт 2.(Если на компьютере есть Git):
      1) Ввести в Git Bash данную строку: git clone https://github.com/EmptyQwazar/Game-Hangman-Group-IT-8-
      2) Далее вести строку для токрытия репозитория: cd Game-Hangman-Group-IT-8-
      3) После запустить игру "Виселица": python Game_Hanmgan_Group_IT-8.py
      4) Наслаждатся крутой игрой
